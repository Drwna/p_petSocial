# 宠物社交圈 (PetSocial) 产品文档

## 1. 产品简介
PetSocial 是一款专为宠物爱好者打造的社交应用，用户可以以宠物的身份创建档案，分享宠物的日常生活，与其他宠友互动（点赞、评论、关注），并支持通过分类浏览发现更多有趣的宠物内容。

## 2. 技术架构
- **前端框架**：UniApp (Vue 3)
- **开发环境**：HBuilderX / VS Code
- **运行平台**：H5 / 微信小程序 / App
- **后端接口**：RESTful API (Base URL: `http://localhost:3001`)

## 3. 功能模块详情

### 3.1 用户与认证模块
- **验证码登录/注册**：
  - 支持手机号发送验证码。
  - 注册时强制填写宠物生日（用于计算年龄）。
  - 支持多账号切换功能。
- **个人资料管理**：
  - 编辑宠物头像、昵称、性别、简介。
  - 个人主页展示：
    - 宠物基本信息（姓名、性别、年龄、简介）。
    - 统计数据（关注数、粉丝数、帖子数）。
    - “我的动态”列表。
    - 删除自己的帖子功能。

### 3.2 首页与内容发现
- **分类浏览**：
  - 顶部支持分类 Tab（全部、猫猫、狗狗等）。
  - **左右滑动切换**：支持通过手势左右滑动直接切换不同分类的内容列表。
- **帖子列表**：
  - 卡片式设计，展示宠物头像、昵称、发布时间、内容文本、多图预览。
  - 底部互动栏：点赞、评论数展示。
  - 点击头像可跳转至对应用户的主页。

### 3.3 内容创作 (发布)
- **图文发布**：
  - 支持上传多张图片（最多9张）。
  - **独立上传策略**：图片先经由独立接口上传，获取 URL 后再提交帖子，提升发布稳定性。
  - 支持选择帖子分类。
  - 必填项校验与友好的加载反馈。

### 3.4 互动系统
- **帖子详情**：
  - 完整展示帖子内容与大图预览。
  - 评论列表：支持查看所有评论，点击评论者头像跳转主页。
  - **评论管理**：
    - 帖子作者可删除其帖子下的任意评论。
    - 普通用户可删除自己发布的评论。
- **关注机制**：
  - 支持在个人主页或帖子详情页关注/取消关注其他宠物。
  - 关注后粉丝数与关注数实时更新。
- **点赞**：
  - 支持对帖子进行点赞/取消点赞，状态即时同步。

### 3.5 用户主页 (查看他人)
- **独立展示页**：
  - 查看其他用户的宠物档案与统计数据。
  - 浏览该用户发布的所有历史帖子。
  - 底部提供“关注/已关注”操作按钮。

## 4. 接口规范 (API)
所有请求均通过 `request.js` 统一封装，并在 Header 中携带 `Authorization: Bearer <token>` 进行身份验证。

| 模块 | 方法 | 路径 | 描述 |
| :--- | :--- | :--- | :--- |
| **账号** | POST | `/api/account/send-code` | 发送验证码 |
| | POST | `/api/account/login-code` | 验证码登录 |
| | POST | `/api/account/register` | 注册新账号 |
| **宠物** | GET | `/api/pet/profile` | 获取当前登录宠物信息 |
| | GET | `/api/pet/profile/:id` | 获取指定宠物信息 |
| | GET | `/api/pet/:id/posts` | 获取指定宠物的帖子列表 |
| **帖子** | GET | `/api/post/list` | 获取首页帖子列表 (支持分页/分类) |
| | POST | `/api/post/create` | 发布新帖子 |
| | POST | `/api/post/delete` | 删除帖子 |
| | GET | `/api/post/:id` | 获取帖子详情 |
| | POST | `/api/post/like` | 点赞/取消点赞 |
| **评论** | GET | `/api/comment/list` | 获取评论列表 |
| | POST | `/api/comment/create` | 发布评论 |
| | POST | `/api/comment/delete` | 删除评论 (软删除) |
| **关注** | POST | `/api/follow/add` | 关注宠物 |
| | POST | `/api/follow/remove` | 取消关注 |
| **上传** | POST | `/api/upload/single` | 单图上传 (支持并发调用) |

## 5. 交互与体验优化
- **手势操作**：首页支持左右滑动切换分类，符合移动端主流操作习惯。
- **即时反馈**：点赞、关注、删除等操作均有 Toast 提示及界面状态的即时更新，无需手动刷新。
- **权限控制**：
  - 删除按钮仅对有权限的用户显示（自己的帖子/评论）。
  - 未登录用户进行互动操作时会自动引导至登录页。
